# Case-study gallery images

Each project's `/work/<slug>` case study shows a **Gallery**. By default it
renders styled placeholder tiles. To use real photos of that job, drop image
files into the matching folder here and rebuild — they replace the placeholders
automatically (no code change). Filenames are shown in sorted order.

- Accepted types: `.jpg` `.jpeg` `.png` `.webp` `.avif` `.gif`
- Recommended: landscape ~4:3, reasonable file size (≤ ~300 KB each)
- After adding files run `npm run build` (or restart `npm run dev`) to pick them up.
